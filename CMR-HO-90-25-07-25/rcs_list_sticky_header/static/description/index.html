Sticky List Headers in Odoo

Reliution's Sticky List Headers in Odoo app enhances the user experience by keeping column headers visible while scrolling through list views. This feature is especially useful when dealing with large datasets in modules like Sales, Inventory, Purchase, HR, and Accounting. It helps users avoid confusion by locking the list headers at the top, making it easier to identify each column as you scroll down.

This module improves efficiency, reduces user errors, and provides a more professional interface across your Odoo environment.

Detailed Features

01 Sticky List View Headers - With the Sticky List View Headers feature, the headers "stick" to the top of the screen as you scroll down the list. Sticky headers also work inside list views embedded within form views.
02 Works Across All List Views - You don’t need to activate it individually per module. Once installed, the sticky header functionality is automatically applied to all list views, regardless of the module or data model.
03 No Configuration Required - As soon as you install the module, the sticky header functionality is automatically activated across all standard list views in your Odoo environment.
04 Performance Optimized - Lightweight design that doesn’t affect page speed.

Set Up

This module requires no additional configuration. Simply install the Sticky List Headers in Odoo module, and open any list view.

Apps >> Sticky List Headers in Odoo >> Activate

Here you can see the header will automatically stick to the top as you scroll.

When working with long lists in Odoo, the default behavior hides the headers once you scroll past them. This causes confusion, especially when dealing with multiple fields or similar-looking data. With Sticky List Headers in Odoo, you can always see the column titles, making data analysis easier and more accurate.
